Kanban para o Projeto TechFit

Nós (Joel e Luiza) vamos escrever este material e conduzir o projeto com Kanban, visualizando o fluxo e limitando o trabalho em progresso (WIP), dividindo as tarefas entre nós.

A Fazer

Módulo de Comunicação

Criaremos um sistema de notificações por e-mail e SMS.

Controle de Acesso

Desenvolveremos um leitor de QR Code para as catracas.

Avaliação Física

Criaremos uma interface para registro de avaliações.

Painel Administrativo

Desenvolveremos a função de gerar relatórios.

Em Desenvolvimento

Interface de Agendamento

Desenvolveremos a interface para a visualização de aulas.

Lógica de Agendamento

Implementaremos a funcionalidade de agendar/cancelar aulas.

Em Revisão

Protótipo de Agendamento

Verificaremos se o protótipo atenderá aos requisitos iniciais.

Concluído

Análise de Requisitos

Finalizaremos o documento de requisitos do projeto TechFit.

Esboço do Sistema

Concluiremos o esqueleto da página web.
