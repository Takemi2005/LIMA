XP no Projeto TechFit

Nós (Joel e Luiza) vamos conduzir este projeto com XP e escrever este material, com foco em qualidade técnica e entregas contínuas.

Pair Programming

Vamos programar em par: enquanto um codificar, o outro revisará em tempo real, e alternaremos os papéis a cada tarefa.

Test-Driven Development (TDD)

Praticaremos TDD: primeiro escreveremos o teste da funcionalidade (por exemplo, o botão "Agendar"), depois implementaremos o código até o teste passar.

Integração Contínua

Integraremos o código com frequência, evitando conflitos e mantendo o sistema sempre funcional.

Pequenas Entregas

Faremos pequenas entregas frequentes (lista de aulas, botão "Agendar" etc.), dividindo as tarefas entre nós e validando a cada incremento.
